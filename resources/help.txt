Usage:
    xenodon [subcommand] [subcommand options]

Subcommands:
help [subcommand or topic]
    Show help for a particular subcommand or topic, or without arguments, this message.

sysinfo
    Detect system information such as connected monitors and video cards.

convert [options] <source> <destination>
    Convert 3D TIFF images to octrees.

render [options] <volume>
    Render a volume.

xorg-multi-gpu
    Information about the config format required for rendering with multiple
    GPUs on X.org.

headless-config
    Information about the config format required for rendering with the
    headless backend.

direct-config
    Information about the config format required for rendering with the
    diret backend.
